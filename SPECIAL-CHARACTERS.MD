| `` ` ``   | Grave accent |
| `()`      | Brackets |
| `[]`      | Square brackets |
| `{}`      | Braces |
| `<>`      | Angle braces |
| `<`       | Less Than |
| `>`       | Greater-than |
| `&`       | Ampersand |
